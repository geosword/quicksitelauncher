Quick Site Launcher

Quick Site Launcher is a streamlined Chrome extension designed for rapid access to your favorite websites using intuitive keyboard shortcuts.

Key Features

  • Lightning-Fast Navigation: Open your most visited sites with a simple keyboard command (e.g., Ctrl+M by default, then a single key for your site).
  • Customizable Shortcuts: Easily define, edit, and manage shortcuts to any URL through the extension's options page.
  • Popup Interface: A clean and simple popup displays your configured shortcuts for quick reference.
  • Open in Current or New Tab: Choose to open sites in your current tab or a new tab (using Ctrl + key).
  • Manifest V3 Compatible: Built with the latest Chrome extension platform standards for improved security, performance, and privacy.

Inspiration

This extension is heavily inspired by the original SiteLauncher Startup Edition. While the original was a fantastic tool, it was based on the older Manifest V2. Quick Site Launcher aims to provide a similar, beloved functionality while being fully compatible with Manifest V3, ensuring it continues to work smoothly with modern Chrome browsers.

Why Manifest V3?

Manifest V3 is the newer standard for Chrome extensions, focusing on enhancing security, privacy, and performance. By building Quick Site Launcher on Manifest V3, we ensure it's future-proof and adheres to Google's latest guidelines for a safer browsing experience.
